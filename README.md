# RC1Computer
8-Bit Relay Computer inspired by Harry Porter and DipDot
# General Information
This Computer, inspired Harry H Porter III, Ph.D, Paul Law, and Youtuber DipDot, has an 8 bit Data- and an 16 Bit address Bus.
